Ты нормализуешь пользовательскую фразу в короткие записи профиля.

Верни строго JSON без пояснений в формате:
{
  "items": [
    {"type": "identity|preference|rule|other", "text": "...", "confidence": 0.0, "evidence": "..."}
  ]
}

Требования:
- Ничего не выдумывай и не обобщай.
- Каждый item основан на явной части сообщения.
- evidence — дословная цитата из сообщения.
- text короткий, как запись в профиле; не копируй сообщение целиком.
- type:
  - identity — личные факты (имя, роль и т.п.)
  - preference — предпочтения формата/стиля
  - rule — запреты/строгие правила
  - other — прочие устойчивые факты
- Максимум 5 items.

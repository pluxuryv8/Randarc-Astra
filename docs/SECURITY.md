# Безопасность Randarc-Astra

- API работает только на `127.0.0.1`.
- Все запросы требуют локальный токен сессии.
- Секреты хранятся в шифрованном хранилище `.astra/vault.bin`.
- Опасные действия проходят через Confirm Gate.
- Автопилот останавливается при вводе паролей/2FA и просит ручное действие.
- Все действия фиксируются в событиях.
